About Jh Godown

Jh Godown is a free, serverless file encoding and sharing platform. It transforms any file into compact Unicode text that can be stored, shared, and decoded back to the original file.

How It Works

Our custom 18-bit encoding engine compresses your files using zlib compression, then encodes the binary data into a string of Unicode characters. This text can be:

  • Copied and pasted anywhere
  • Stored in Google Sheets (up to 500,000 rows per sheet)
  • Shared via a simple Post ID
  • Decoded back to the exact original file

Technology Stack

Built entirely on free-tier services with zero hosting costs:

  • Frontend: Pure HTML/CSS/JS hosted on GitHub Pages
  • Backend: 6 Google Apps Script Web Apps
  • Database: Google Sheets (registry + storage)
  • Compression: pako.js (zlib port)
  • Security: SHA-256 hashing, client-side sessions
  • Notifications: Telegram Bot API

Features

  • Encode any file format to compact Unicode text
  • Web Worker background processing (no UI freeze)
  • Offline encoding support
  • Social feed with infinite scroll
  • User profiles with analytics
  • Bookmark and organize posts
  • Dark/Light theme toggle
  • QR code sharing
  • PWA support (install on mobile)
  • Rate limiting and spam protection
